# challenge-alura-python
Análisis de ventas
Este proyecto tiene como objetivo analizar el rendimiento de las cuatro tiendas de la cadena Alura Store. 

Se evaluarán métricas clave como:
- Ingresos totales
- Categorías más vendidas
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo promedio de envío

A partir de este análisis se realizará una recomendación sobre qué tienda debería vender el Sr. Juan para invertir en un nuevo emprendimiento.
