# challenge-alura-python
Análisis de ventas
Este proyecto tiene como objetivo analizar el rendimiento de las cuatro tiendas de la cadena Alura Store. 

Se evaluarán métricas clave como:
- Ingresos totales
- Categorías más vendidas
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo promedio de envío

A partir de este análisis se realizará una recomendación sobre qué tienda debería vender el Sr. Juan para invertir en un nuevo emprendimiento.
Después de analizar los datos de ventas de las cuatro tiendas de Alura Store, se evaluaron diferentes indicadores como los ingresos generados, el rendimiento de los productos, las calificaciones de los clientes y los costos de envío.

El análisis de ingresos muestra que:

Tienda 1 generó aproximadamente 1,150,880,400 en ingresos.

Tienda 2 generó aproximadamente 1,116,343,500.

Tienda 3 generó aproximadamente 1,098,019,600.

Tienda 4 generó aproximadamente 1,038,375,700.

A partir de estos resultados, se observa que la Tienda 4 presenta el menor nivel de ingresos en comparación con las demás tiendas. Esto indica un menor rendimiento comercial dentro de la cadena.

Por lo tanto, se recomienda que el Sr. João venda la Tienda 4, ya que es la que presenta el desempeño más bajo. Esta decisión permitiría concentrar recursos en las tiendas más rentables y facilitar la inversión en un nuevo emprendimiento.
